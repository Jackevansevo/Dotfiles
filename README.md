# Dotfiles

![Image](EpXhHSiWMAAoSeB.png "TFW")

    git clone --recurse-submodules git@github.com:Jackevansevo/Dotfiles.git

If you forget to use `--recurse-submodules` when cloning:

    git submodule update --init

See the [git submodules documentation](https://git-scm.com/book/en/v2/Git-Tools-Submodules)
